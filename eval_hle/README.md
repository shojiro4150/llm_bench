# Humanity's Last Exam 評価コード
```bash
llm_bench/
├── conf/         # 評価設定（config.yaml）
├── eval_hle.sh   # 推論→採点を自動実行するスクリプト
├── predictions/  # 生成結果（実行後に作成）
├── judged/       # 採点結果（実行後に作成）
├── vllm.log      # vLLMサーバーログ（実行後）
├── predict.log   # 推論ログ（実行後）
└── judge.log     # 採点ログ（実行後）
````
